# edm-sci

edm-sci is a python package for calculating the Disruption Index and Embedding Disruptiveness Index of a paper or patent using a citation network.

The original node2vec code, which implements the directional skip-gram algorithm, is done by @skojaku. You can find the original code [here](can be found at https://github.com/skojaku/node2vec ). edm-sci is a modified version of that code, adapted specifically to calculate the disruption index and embedding didsruptiveness measure.

## Installation

install the latest version of edm-sci

`pip install --upgrade edm-sci`
